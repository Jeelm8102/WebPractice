position: static;

An element with position: static; is not positioned in any special way; it is always positioned according to normal flow of the page:

This division element has position: static;

position: relative;

An element with position: relative; is not positioned relative to its normal position:

This division element has position: relative;

position: absolute;

An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed):

This div element has position: relative;
This div element has position: absolute;
